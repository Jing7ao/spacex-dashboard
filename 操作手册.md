> 版本：v3.3 · 2026-06-10 | 访问地址：`http://localhost:8080`

---

## 一、系统概览

四个页面：

| 页面 | URL | 功能 |
|------|-----|------|
| **催化日历** | `/index.html` | 事件日历 + 供应链新闻(自动情感分析) + 标的筛选 |
| **市场监控** | `/monitor.html` | 23板块监控 + 评分/量价/主力意图/交易记录 |
| **预测面板** | `/predict.html` | 生态轮动 + 买入检查清单 + 退出信号 |
| **模型总览** | `/model-overview.html` | 物理开关股价 + 生态对比 + 证据缺口 |

### 启动

```
双击 启动面板.bat → 自动更新数据 → 启动代理 → 浏览器打开 http://localhost:8080
```

### 盘前检查流程（每天 2 分钟）

1. 打开 `/monitor.html` → 看顶部 ⚠️ 开盘预警（美股映射+利空新闻）
2. 看 主力意图面板 → 谁在建仓、谁在撤退
3. 看 🚨 预警面板 → 超买/超卖/止损提醒
4. 下单时 → 弹窗自动过买入检查清单
```

## 二、催化日历 + 标的分析（index.html）

### 2.1 页面结构

```
[📊 量化模型]  [📊 市场监控]

  📅 催化日历  │  📊 标的分析    ← 标签页

**默认打开催化日历**，从事件出发驱动分析。

### 2.2 标签1：📅 催化日历（默认）

#### 六大板块

| 板块 | 内容 | 使用方式 |

| 🟢 财报季 | 年报/季报截止日 | 截止前确认业绩 |
| 🟠 持仓解禁日 | 来自 risk-check 的持仓解禁数据 | >20% 流通盘评估减仓 |
| 🔵 产业事件 | CES、NVIDIA GTC、COMPUTEX、特斯拉 AI Day、华为大会、台积电大会 | 会前布局供应链标的 |
| 🟣 指数调整 | MSCI、科创50、沪深300 调整 | 关注纳入/剔除标的 |

#### 颜色标记

| 标记 | 含义 |
|------|------|
| 🔴 红色左边框 | 3 天内紧急 |
| 🟡 黄色左边框 | 2 周内 |
| 🟢 绿色左边框 | 远期 |

#### 事件操作

| 操作 | 效果 |
|------|------|
| **点击事件行**（如"NVIDIA GTC →分析"） | 切到分析页，筛选该事件关联的全部标的 |
| **点击个股标签**（如"[中际旭创]"） | 切到分析页，高亮该个股 + 显示供应链详情 |
| **点击政策/财报事件** | 切到分析页，显示全板块综合评估 |

产业事件下方会直接显示关联标的的彩色标签，一目了然。

#### 评分公式

```
综合评分 = RSI(20%) + 均线(15%) + 量能(15%) + 布林(10%) + 供应链卡点(25%) + 资金流(15%)

扣分项:
  PE>300 或 PE=0  :  -10分
  PE>200          :   -5分
  风险数据(解禁/质押/商誉) :  -0~-25分
  概念股(scpS≤2)   :  -10分
  MA60破位         :  ×0.85

✓ = 评分≥65（优质标的）
```

| 星级 | 分数 | 标准 | 示例 |

## 四、日常操作流程

### 开盘前（9:00）

```
1. 打开 http://localhost:8080
2. 默认显示催化日历 → 查看今日事件
3. 如有产业事件（如 NVIDIA GTC） → 点击 → 查看关联标的评分
4. 如有解禁日 → 评估持仓是否需要减仓
```

### 盘中（9:30-15:00）

```
1. 从日历点击事件 → 切到标的分析看评分排名
2. 评分≥65 + 卡点≥8 → 高优先级考虑
3. 切到 monitor.html 查看行业详情 + 资金流向
4. 决策后 → 在券商 APP 执行交易
```

### 收盘后（15:00+）

```
1. monitor.html → 持仓面板更新盈亏
2. 记录交易（📒 交易记录）
3. 每周：node risk-check.js 更新风险数据
   或访问 /api/risk 自动触发（>6h 自动刷新）
```

## 五、风险速查

| 风险类型 | 检测工具 | 阈值 | 行动 |
|----------|----------|------|------|
| 解禁 | 催化日历 🟠 + risk-check | >20% 流通盘 | 减仓避险 |
| 质押 | risk-check | >50% | 不碰 |
| 商誉 | risk-check | >30% 净资产 | 警告 |
| 减持 | risk-check | 近期有减持 | 关注 |
| PE 极端 | 评分自动扣分 | >300 | -10 分 |
| MA60 破位 | 评分自动扣分 | 价格<MA60 | ×0.85 |
| 止损 | 持仓面板 | -8% | 立即卖出 |
| 止盈 | 持仓面板 | +15% | 分批卖出 |
