# 📊 量化模型 操作手册

> 版本：v5.1 · 2026-06-25 | 访问地址：`http://localhost:8080`

## 一、系统概览

四个页面：

| 页面 | URL | 功能 |
|------|-----|------|
| **催化日历** | `/index.html` | 事件日历 + 供应链新闻(自动情感分析) + 标的筛选 |
| **市场监控** | `/monitor.html` | 11生态监控 + 评分/量价/主力意图/交易记录 |
| **预测面板** | `/predict.html` | 生态轮动 + 买入检查清单 + 退出信号 |
| **模型总览** | `/model-overview.html` | 物理开关股价 + 生态对比 + 证据缺口 |

### 启动

```
双击 启动面板.bat → 自动更新数据 → 启动代理 → 浏览器打开 http://localhost:8080
```

### 盘前检查流程（每天 2 分钟）

1. 打开 `/monitor.html` → 看顶部 ⚠️ 开盘预警（美股映射+利空新闻）
2. 看 主力意图面板 → 谁在建仓、谁在撤退
3. 看 🚨 预警面板 → 超买/超卖/止损提醒
4. 下单时 → 弹窗自动过买入检查清单
```

#### 评分公式（v4.1 统一 IC 优化版）

```
综合评分 = 趋势(35%) + 均线(20%) + RSI(15%) + 量能(15%) + 布林(10%) + 资金流(5%)

权重来源: 全局IC回测（趋势 -0.370, 均线 +0.079, RSI -0.057, 量能 +0.123, 布林 -0.045）
供应链作为前置筛选层（生态选择），不计入评分权重

扣分项:
  PE>300 或 PE=0  :  -10分
  PE>200          :   -5分
  风险数据(解禁/质押/商誉) :  -0~-25分

信号阈值: ≥75 强烈买入 | ≥60 买入 | ≥40 持有 | ≥25 卖出 | <25 强烈卖出
```

## 四、日常操作流程

### 开盘前（9:00）

```
1. 打开 http://localhost:8080
2. 默认显示催化日历 → 查看今日事件
3. 如有产业事件（如 NVIDIA GTC） → 点击 → 查看关联标的评分
4. 如有解禁日 → 评估持仓是否需要减仓
```

### 盘中（9:30-15:00）

**开盘前 30 分钟（9:30-10:00）：只看不动**

```
1. 观察竞价和开盘方向，不急于下单
2. 信号驱动入场：等 30 分钟后再判断（策略 §3 规定）
3. 事件驱动入场：竞价期可操作，但必须提前一天确定标的+限价
4. 跳空 > 3% 的票当天不追
```

**10:00 后：按信号执行**

```
1. 切到 monitor.html → 确认四灯状态（评分/量价/资金/供应链）
2. 四灯全绿 → 准备入场
3. 检查 trend 状态（MA5/MA20）→ 确定限价偏移方向
4. 在券商 APP 下限价单：参考价 ± 偏移%
```

**限价偏移速查**：

| 趋势 | 判定 | 偏移 |
|:--:|------|:--:|
| 📈 涨 | 现价 > MA5 > MA20 | 参考价 **+1%** |
| 📉 跌 | 现价 < MA5 或 MA5 < MA20 | 参考价 **-1%** |
| ➡️ 震 | 均线缠绕 | 参考价 **±0%** |

> 主力净流入 > 5000 万且量比 > 1.5：涨势偏移放宽到 +1.5%
> 竞价期（仅事件驱动）：不看 MA，看竞价方向：连续升 → +1%，连续跌 → -1%
> 详见策略说明书 §3.2.1

```
5. 挂单后不要盯盘频繁改价——限价单的价格纪律比成交速度更重要
6. 买入后立即在持仓面板记录（monitor.html → 📦持仓 → +添加）
```

### 收盘后（15:00+）

```
1. monitor.html → 持仓面板更新盈亏
2. 记录交易（📒 交易记录）
3. 每周：node risk-check.js 更新风险数据
   或访问 /api/risk 自动触发（>6h 自动刷新）
```

## 五、风险速查

| 风险类型 | 检测工具 | 阈值 | 行动 |
|----------|----------|------|------|
| 解禁 | 催化日历 🟠 + risk-check | >20% 流通盘 | 减仓避险 |
| 质押 | risk-check | >50% | 不碰 |
| 商誉 | risk-check | >30% 净资产 | 警告 |
| 减持 | risk-check | 近期有减持 | 关注 |
| PE 极端 | 评分自动扣分 | >300 | -10 分 |
| MA60 破位 | 评分自动扣分 | 价格<MA60 | ×0.85 |
| 止损 | 持仓面板 | -8% | 立即卖出 |
| 止盈 | 持仓面板 | +15% | 分批卖出 |
