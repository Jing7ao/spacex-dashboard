## 快速开始

### 方式一：直接打开（推荐）

双击 `index.html` 即可在浏览器中打开。

> 如果行情数据加载失败，可能是浏览器 CORS 限制。使用方式二。

### 方式二：本地代理（解决 CORS）

```bash
# 确保已安装 Node.js
node proxy.js
# 然后浏览器打开 http://localhost:8080/index.html
```

### 方式三：浏览器 CORS 插件

安装 Chrome/Edge 扩展 "Allow CORS: Access-Control-Allow-Origin"，然后刷新页面。

- **个股风险标签**和风险提示

### 📐 统一六维评分（v4.0）

所有生态使用同一套评分标准，基于全局 IC 分析确定权重：

| 维度 | 权重 | 数据来源 | IC | 方向 |
|------|:----:|----------|:--:|:--:|
| 趋势强度 | 35% | MA20/MA60偏离 | -0.370 | 🔄 强趋势→反转 |
| RSI(14) | 15% | K线收盘价 | -0.057 | 🔄 超买→回调 |
| MA趋势 | 20% | MA5/MA20 | +0.079 | → 顺势 |
| 成交量 | 15% | 当日量/20日均量 | +0.123 | → 放量确认 |
| 布林带 | 10% | 20日布林带 | -0.045 | 🔄 极端→回归 |
| 资金流向 | 5% | 主力净流入 | — | → 辅助确认 |

> 因子 IC 来自 2025.12-2026.05 六个月全局分析。趋势强度和 RSI 方向稳定（6/6 月一致），是评分核心驱动力。

**信号解读：**

| 评分区间 | 信号标签 | 操作 |
|:--------:|----------|------|
| 75-100 | 🔥 强买入 | 可买入/加仓 |
| 60-74 | 📈 偏买入 | 可试仓 |
| 40-59 | ➡️ 观望 | 持有不动，不新买 |
| 25-39 | ⚠️ 偏卖出 | 准备卖出 |
| 0-24 | 🚨 强卖出 | 立即清仓 |

### 🚨 实时预警
自动触发条件：
- RSI > 75（超买）
- RSI < 25（超卖）
- 单日跌幅 > 7%
- 放量下跌（量比 > 2 且跌 > 3%）
- 主力资金净流出 > 1 亿
- 跌破 20 日均线

### 📰 资讯聚合
每 5 分钟聚合以下关键词的最新资讯：
- 商业航天、SpaceX、卫星互联网、低轨卫星、星舰

### 🔍 信息源状态
自动监控数据源连接状态。绿色=正常，黄色=降级/延迟，红色=不可用。

**每日手动核查清单**（需自行打开链接查看）：
- SEC EDGAR — SpaceX SPCX 公告
- 巨潮资讯网 — 持仓公司公告（减持/定增/业绩预告）
- 工信部 / 国家航天局 — 商业航天政策
- SpaceX 官推 — 星舰试飞 / Starlink 用户数据

## 🎯 每日操作流程

```
① 持仓评分 < 40 → 卖出
② 有预警红色 → 立即处理
③ 评分 > 60 且有仓位 → 试仓买入 1-2 只
④ 评分 40-60 → 不动
```

**核心规则：评分 < 40 之前，不看盈亏，不动。**

## 技术栈

- 面板：HTML + CSS + JavaScript（单文件 `monitor.html`）
- 评分：统一六维加权 + 18因子库（`eco_factors.py` 可重跑 IC 分析）
- 回测：`backtest.py`（支持简单 OOS 和滚动回测两种模式）
- 代理：`proxy.js`（Node.js，解决浏览器 CORS）
- 数据：东方财富公开 API、腾讯 K 线 API、Yahoo Finance API
- 存储：localStorage（设置/持仓/交易记录持久化）
